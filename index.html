Difference between Object.freeze(), Object.seal() and Object.preventExtensions()

The Object.freeze() method

  • This method completely freezes an object.
  • It means that if an object is frozen, then we can’t add a new property to it,
  • can’t update any existing property value,
  • and we also can’t delete any existing properties.
  • To check if an object is frozen, we can use the Object.isFrozen() method.
